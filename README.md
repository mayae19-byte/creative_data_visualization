# creative_data_visualization

Creative Data Visualization: Proteome Sonification


This project presents a creative, nontraditional data visualization of proteomics data by transforming protein abundance changes into sound. Instead of relying solely on visual plots, this approach uses sonification to represent molecular differences associated with alcohol (EtOH) exposure.

Each protein is translated into a musical note, allowing patterns in large-scale biological data to be experienced through an auditory medium.
